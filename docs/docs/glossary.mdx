# Glossary

- Channel - Its basically the representation of a DAPP on CPNS. Any DAPP developer would also own the corresponding channel for it on CPNS. They can use this channel to send notficiations to users of their DAPP by asking them to subscribe to the channel.
- Subscriber - Any user when they subscribe to a channel, becomes a subscriber for that channel. Only subscribers to a channel can be sent notifications.
- Private Notification - This is a special type of notification that is encrypted before it even makes it onto the chain. This uses public keys of users published on chain to encrypt messages.
